■ BoardFlow

Connect your GitHub repository to track shared tasks as GitHub Issues — visible to your whole team.

Quick setup:
1. Go to github.com → Settings → Developer settings → PATs → Tokens (classic)
2. Click Generate new token (classic)
3. Check the top-level repo scope
4. Set expiration to No expiration
5. Click Generate token and copy it below
Stored only in your browser's localStorage. Never sent anywhere except GitHub's API.
